# AI-Pharmacy-POS

An AI enabled Point of sale for Pharmacies that can read physical prescriptions, calculate invoices, manage inventory, and more.

## Special Features

- Ai Prescription OCR
- Expiry Date tracking Low stock tracking
- Payment reminders, financial analytics
- Loyalty Management Prescription History.

## Quick Setup Guide

For detailed instructions on how to set up and deploy the application (both frontend and backend), please refer to our [Deployment Guide](deployment_guide.md).

## Key Features

- **AI Prescription Processing:** Automatically extract medicines, frequencies, and amounts from uploaded prescription images using Google's Gemini AI.
- **Advanced Point of Sale (POS):** Complete checkout capabilities, draft saving for pending sales, voiding, real-time product searches, and PDF receipt generation.
- **Inventory & Batch Management:** Track products across batches with First-Expired-First-Out (FEFO) logic, alerting for low stock or near-expiry batches. Handle Goods Receipt Notes (GRN) and manage suppliers.
- **Patient Management & CRM:** Securely manage patient profiles with PII encryption. Calculates automatic discounts for seniors and loyalty customers. Fully compliant with patient opt-out actions.
- **Financial Analytics & Accounting:** Complete dashboard charting revenue, COGS, operating expenses, and net profit. Also supports basic employee payroll and supplier check management.
- **Audit Logging:** Exhaustive administrative tracking of system actions, logging every sensitive operation with CSV export capabilities.
